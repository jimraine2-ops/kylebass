## [기계적 직결 진입] 30억 수급 × EMA 역추세 최종 집행 지침

### 4대 AND 게이트 (Hard-Criteria)
1. **수급 검증**: 과거 20거래일 평균 거래대금 30억원($2.22M) 이상 유지
2. **이격도 확정**: 현재가 < EMA25 × 0.95 (5%+ 이격)
3. **추세 확정**: 25봉 연속 하락 + 현재 음봉(하락봉)
4. **에너지 확정**: 체결강도 90%+

### Free-Tier Hack (15분 시차 극복)
- P_buy = P_delayed × (1 - 0.015~0.02)
- 15분 뒤 실제 저점에 미리 그물(Limit Order) 배치

### Infinite-Gain Loop
- 매수 즉시 본절보호 +0.2% 가동
- 3% 익절 목표 (EMA25 회귀), 체결강도 변화 시 2~2.5% 유연 확정
- 익절 완료 → ₩100만 리셋 → Round N 무한 루프
